# 🗂️ File Organizer in Python

This project is a simple yet powerful Python script that organizes files in a given folder into categorized subfolders based on their file extensions.

## 🚀 Features

- Automatically sorts files into folders:
  - 📷 Images: `.jpg`, `.jpeg`, `.png`, `.gif`
  - 📄 Documents: `.pdf`, `.docx`, `.txt`, `.pptx`
  - 🎥 Videos: `.mp4`, `.mkv`, `.mov`
  - 🎵 Music: `.mp3`, `.wav`
  - 📦 Archives: `.zip`, `.rar`, `.7z`
  - 💻 Scripts: `.py`, `.js`, `.html`, `.css`
  - 📊 Spreadsheets: `.xls`, `.xlsx`, `.csv`
  - 📁 Others: anything else!

- Automatically creates folders if they don’t exist
- Handles both files and nested directories

## 🧰 Requirements

- Python 3.x
- No external libraries needed (uses built-in `os` and `shutil` modules)

## 📝 How to Use

1. Clone or download this repository.
2. Open the folder in VS Code or your terminal.
3. Run the Python file:

```bash
python app.py
